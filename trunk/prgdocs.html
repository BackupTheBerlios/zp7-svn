Zpěvníkátor - programátorská dokumentace

Balíčky

  • anchors - kotvy pro features - definice rozhraní
  • browse - jednoduchá definice dialogů a oken (rozmístění, eventy)
  • config - konfigurace - konfigurační soubory hotkeys.py, fonts.py, editor konfigurace
  • database - přístup k databázi písní, filtrování
  • dbview - zobrazovač databáze
    • dbvpanel - panel "Databáze" v hlavním okně
    • dbgrid - obecná databázová tabulka - užívá se pro skupiny a písně
    • songview - zobrazovač textu písně (wxControl)
  • desktop - definice hlavního okna, většina jeho obsahu je ovšem v pluginech
  • format - formátování písní
  • images - obrázky - převedáděny automaticky do pythonovských zdrojáků
  • internet - přístup na server zpěvníkátoru
  • interop - práce se zprávami, definice pluginovacího principu anchor-feature (v anchor.py)
  • printing - práce s tiskárnou
  • songbook - práce se zpěvníkem
    • sbpanel - panel "Zpěvník" v hlavním okně
    • songbook - objekt zpěvníku
    • realsb - reálný typ zpěvníku - v závislosti na konkrétní tiskárně
    • stdfeatures - definice základních features zpěvníku (definice rozhraní v anchors/songformat.py), např. pro tisk záhlaví pisně, tisk čar mezi písněmi
    • paging - definice LogPages
    • distribalg - definice distribučních algoritmů (list of panegrp=>LogPages), správný distribalg je vybrán pomocí stdfeatures.py/StdDistribAlg (např.)
    • sbtype - definice typu zpěvníku
  • songtool - transpozice
  • utils - různé utility
